# React Context Todo App (CRUD)

A modern Todo application built with React and Context API. This project demonstrates full CRUD functionality (Create, Read, Update, Delete) with clean UI and global state management.

---

## 🚀 Features

- Add new tasks
- Edit tasks
- Delete tasks
- Context API for state management
- Responsive UI
- Clean and simple design

---

## 🧠 Concepts Used

- React Context API
- CRUD Operations
- State Management
- React Hooks
- Component Architecture

---

## 🛠 Tech Stack

- React
- JavaScript
- Context API
- CSS
- Vite
